NCR Receipt Printer 7167
=======================

Prerequisites
-------------

1. node.js

2. npm

Execution Instruction
---------------------

1.
`npm install`

1.
`npm start`

### Endpoint:

Send to `localhost:3000/print/message` some json in the format:

```
{ "message": "I would really like this message to be printed please." }
```
